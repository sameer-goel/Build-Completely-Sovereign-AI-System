🦀 Takeaway

Every AI coding agent — Claude Code, Cursor, Copilot — is this same pattern: a brain that thinks, tools that do, and a loop that repeats. In the next act, you'll talk to the brain directly. By Act 4, you'll have built your own Local Clawd.

What you're looking at

This is the highest-level view of how Claude Code works. Three pieces, one loop. Everything else in this course is zooming into these boxes.

The 3 pieces

  • You — type in the terminal
  • AI Agent — orchestrates everything
  • Local LLM — the brain

Key insight

The agent doesn't just chat — it uses tools. It reads files, writes code, runs commands. The LLM decides when to use which tool.
